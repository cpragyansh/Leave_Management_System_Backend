

# Leave Management System - Backend API

This is the backend API server for the Leave Management System. It is built using Node.js, Express, and MongoDB. It handles user authentication (JWT) and all operations related to applying for and reviewing leave requests.

## Setup and Installation

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install npm dependencies:
   ```bash
   npm install
   ```

4. Run the seed script to clean the database and create test users:
   ```bash
   node seed.js
   ```
5. Start the API server:
   ```bash
   npm start
   ```
